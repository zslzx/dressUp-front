# dressUp-front
generate image of dressed person

## install
install node and npm first

then run
```
npm install
```
to install node modules

## run
edit package.json:
set "proxy" to the url of dressUp backend

run:
```
npm start
```

visit `localhost:3000` to use it

## build
run:
```
npm run build
```
then copy files in `./build` to `...(backend dir)/public`
